## 💡 使用技巧

### Frontend Design
- **明确描述风格**: "现代简约风格" / "赛博朋克风格" / "极简主义"
- **提供上下文**: 说明是什么类型的应用（电商、社交、工具等）
- **指定技术栈**: React / Vue / 原生 HTML+CSS

### Playwright 测试
- **描述用户流程**: "用户打开页面 → 点击登录 → 输入账号密码 → 提交"
- **指定断言**: "验证页面标题是..." / "确认按钮可点击"
- **处理异步**: Playwright 会自动等待元素加载

### Brainstorming
- **开放式提问**: 不要限制太多，让 Claude 提供多种方案
- **迭代讨论**: 基于回答继续深入讨论
- **要求对比**: "比较这几种方案的优缺点"
