流程

        

        

        

      

      

        流程说明 / 交接给新项目
        
    
          
  1. 一键入口: 点击“一键注册 + token + 设置支付码”，入口函数是 runOneClick()。
    2. 
          
  2. SMSBower: smsGetNumber() 使用 getNumberV2 获取号码，默认 service=ni、country=6；smsWaitCode() 轮询验证码。
    3. 
          
  3. 注册探测: runFirstStage() 先请求 /goto-auth/login/methods，只有响应错误码 auth:error:user:not_found 才切换 flow=signup 继续。
    4. 
          
  4. 发送注册 OTP: runSendOtp() 串行执行 /cvs/v1/methods 和 /cvs/v1/initiate，保存 verification_id 与 otp_token。
    5. 
          
  5. 验证注册 OTP: runVerify() 请求 /cvs/v1/verify，拿到 verification_token。
    6. 
          
  6. 设置用户 + token: runAccountList() 调 /v7/customers/signup 设置昵称并拿 signup token，随后自动执行 runToken() 用 /goto-auth/token 换最终 access_token。
    7. 
          
  7. 设置支付码: runPinMethods()、runPinInitiate()、runPinVerify()、runPinSetup() 分别完成支付码验证码 methods、发码、验码、最终 PIN 设置。
    8. 
          
  8. 签名和日志: 所有 GoPay POST 统一走 signedPost()，里面生成 x-m1、x-e1、x-e2；请求明细写入“请求日志”，业务进度写入“流程日志”。
    9.
